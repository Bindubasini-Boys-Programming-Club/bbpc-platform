#### Badge

| Prop | Type | Default | Description |

| variant | "default" \| "secondary" \| "destructive" \| "outline" \| "success" \| "warning" \| "info" | "default" | Visual style variant |
| href | string | - | Renders badge as a link |
| class | string | - | Additional CSS classes |
