# BBPC — Bindubasini Boys Programming Club

The official website of **Bindubasini Boys Programming Club (BBPC)**.

BBPC is a student-led programming community dedicated to learning, innovation, collaboration, and real-world software development. This platform serves as the central hub for our members, events, projects, resources, and club activities.

---

## About

This website provides:

- Member Management
- Authentication System
- Club Events
- Project Showcase
- Blog & Announcements
- Gallery
- Learning Resources
- AI Assistant
- Member Dashboard
- Responsive Design
- Dark & Light Theme

---

## Tech Stack

- Astro
- React
- TypeScript
- Tailwind CSS
- SQLite (Development)
- Vercel (Deployment)

---

## Installation

Clone the repository:

```bash
git clone https://github.com/Bindubasini-Boys-Programming-Club/bbpc-platform.git
```

Go to the project folder:

```bash
cd bbpc-platform
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build the project:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

---

## Deployment

This project is intended to be deployed on Vercel.

For production authentication and persistent data, a managed database such as PostgreSQL, Supabase, or Neon can be integrated in the future.
